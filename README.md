# Afroza Akter Academic Website

This is a responsive academic portfolio website designed for GitHub Pages.

## Files

- `index.html` — website content
- `style.css` — website design
- `script.js` — mobile navigation and automatic footer year
- `cv.pdf` — downloadable curriculum vitae

## Publish on GitHub Pages

1. Upload all files to the root of the `afroza-akter-cuet.github.io` repository.
2. Keep the filenames unchanged.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select **main** and **/(root)**.
6. Save.

The website will be available at:

`https://afroza-akter-cuet.github.io`

## Update later

Edit `index.html` to change text, `style.css` to change appearance, and replace `cv.pdf` with a newer CV using the same filename.
